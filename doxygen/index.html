Features

  • Flexible mesh data structure

  • Large Scale Parallelization
  • Multi-physics
  • Spectral Finite Difference Solver
  • Residual Distribution Solver
  • Finite Element Solver

Users

There are several ways for users to interact with COOLFluiD and it's modules

  • Cross-platform Graphical User Interface, providing client-server interaction
  • Python scripting
  • coolfluid-command for command-line interpretion and scripting
  • standalone executables
