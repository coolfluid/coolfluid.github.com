enum LogTag

Special message tag.

For now, the only available tag is ENDLINE.

Enumerator
ENDLINE 

Indicates the end of a message.

This tag must be used to flush stream buffers and must end each log message to guarantee that the stream buffers are flushed.

If the pointer is NULL or does not exist in the forwarder list, nothing is done.
