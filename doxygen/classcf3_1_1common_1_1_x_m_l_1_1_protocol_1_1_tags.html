Detailed Description

Provides tags related to the XML protocol The first word of each function indicates the place it should be used in the XML tree:

  • node_xxx : gives a node name
  • attr_xxx : gives an attribute name
  • key_xxx : gives a value for 'key' attribute
  • node_type_xxx : gives a value for 'type' key in a 'frame' node
    Author
    Quentin Gasper
